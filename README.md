# z_movie

#### 介绍
本项目为django教学项目[video_one_teach](https://gitee.com/glittering/video_one)的代码。
看教学文档的请跳转。


#### 安装教程

1.  pipenv install
2.  pipenv run python manage.py makemigrations  
3.  pipenv run python manage.py migrate
4.  pipenv run python manage.py runserver

#### 使用说明

部署服务器的话
1.  gunicorn
2.  supervisor
3.  nginx
